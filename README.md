# FoodApplication
**Тестовое задание**
____
## Описание
Приложение для просмотра и заказа еды. Данные берутся с сервера и отображаются в приложении.
## Используемые технологии
- Kotlin
- MVVM, Fragments
- RecyclerView
- Retrofit, Glide
- Coroutines
- Dependency Injection - Koin
- Модули - app, data
